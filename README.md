# 📂 Repository Contents

This repository contains the following internship tasks:

## 🎮 Task 1 – Hangman Game

### Overview
A console-based word guessing game where the player attempts to guess a hidden word by entering one letter at a time before running out of attempts.

### Features
- Random word selection
- Letter-by-letter guessing
- Incorrect attempt counter
- Win/Lose conditions
- User-friendly gameplay

### Python Concepts Used
- Variables
- Loops
- Conditional Statements
- Strings
- Lists
- Functions
- Random Module

---

## 📈 Task 2 – Stock Portfolio Tracker

### Overview
A simple stock portfolio tracker that calculates the total investment value based on user-entered stock quantities and predefined stock prices.

### Features
- Stores stock prices using dictionaries
- Accepts user input
- Calculates total portfolio value
- Displays investment summary

### Python Concepts Used
- Dictionaries
- Loops
- User Input
- Arithmetic Operations
- Functions

---

## 🤖 Task 3 – Basic Chatbot

### Overview
A rule-based chatbot capable of responding to predefined user inputs through simple conversational logic.

### Features
- Greets the user
- Responds to common questions
- Interactive conversation
- Exit option

### Python Concepts Used
- Functions
- if-else Statements
- Loops
- String Manipulation
- User Input
